## Upgrade notes / breaking changes

- **Major UI Change**: Streamlit interface completely replaced with Next.js frontend
- **New Dependencies**: Requires Node.js 18+ and npm for frontend development
- **API Architecture**: New FastAPI backend replaces direct Python function calls
- **Database Schema**: Extended Neo4j schema with conversation and entity support
- **Configuration**: New environment variables and feature flags added
- **Deployment**: Docker Compose setup now includes both backend and frontend services

## Migration path

1. **Backup existing data**: Ensure Neo4j database is backed up
2. **Update dependencies**: Follow SETUP_V2.md for new Python and Node.js requirements
3. **Configure environment**: Update .env file with new variables and feature flags
4. **Database compatibility**: Existing documents and chunks work without migration
5. **Gradual rollout**: Both v1.x and v2.0 can run simultaneously during transition
